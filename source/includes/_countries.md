# Countries

## Get All Countries

```json
[
    {
        "id": 1,
        "name": "Argentina",
        "liveGames": 2,
        "totalGames": 5
    },
    {
        "id": 2,
        "name": "Brazil",
        "liveGames": 2,
        "totalGames": 5
    }
]
```

This endpoint retrieves all countries.

### HTTP Request

`GET https://ws.365scores.com/web/countries`

<aside class="notice">
Don't forget general parameters.
</aside>
