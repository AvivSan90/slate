# Countries

## Get All Countries

```json
[
    {
        "id": 1,
        "name": "Brazil",
        "nameForURL": "brazil",
        "liveGames": 2,
        "totalGames": 5,
        "sportTypes": [1, 2, 8, 12, 13]
    },
    {
        "id": 1,
        "name": "Argentina",
        "nameForURL": "argentina",
        "liveGames": 2,
        "totalGames": 5,
        "sportTypes": [1, 2, 8, 13]

    }
]
```

This endpoint retrieves all countries.

### HTTP Request

`GET https://ws.365scores.com/web/countries`

<aside class="notice">
Don't forget general parameters.
</aside>
